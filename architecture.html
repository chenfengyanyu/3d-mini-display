MyBetterDisplay Architecture

Claude Code 实时状态可视化系统 · macOS 虚拟屏管理工具

核心数据流

  • • Claude Code 触发 Hook 事件
  • • set-state.sh 解析 JSON 后 curl POST 推送
  • • monitor.js 内存状态机接收并更新状态
  • • index.html 每 250ms 轮询 /api/status
  • • postMessage 驱动 iframe 主题动画

关键设计决策

  • • HTTP 推送取代文件轮询，状态切换精准无竞态
  • • busy 到来立即取消 3s 倒计时，绝不误触绿灯
  • • 前端零计时器，严格跟随服务端状态
  • • Hooks 幂等注入，重复执行安全
  • • 仅限 localhost，无网络暴露风险

技术栈 & 依赖

  • • Node.js 16+（monitor.js HTTP 服务 + 状态机）
  • • 纯 Canvas + requestAnimationFrame（16 主题）
  • • Bash + curl（Hook 执行器）
  • • BetterDisplay 可选（480×640 虚拟屏）
  • • macOS 12.0+ 专属工具链
